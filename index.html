The gold path is computed, not drawn to look good.

The simple game uses lateral offset as the state and bounded sideways correction as the input. For every current offset, the code evaluates the exact constrained finite-horizon feedback law. This preserves a true optimal trajectory while keeping the controls understandable.

Technical interpretation after the game

Cost, continuous-time updates, and a map of optimal laws.

Exact analytical controller

The cost is tangible

Error and effort compete.

Being far from the center accumulates tracking cost. Large sideways commands accumulate effort cost. The optimum balances both over a fixed horizon.

—

Continuous time matters

The best command can change between updates.

The continuous controller leaves the input limit at the exact boundary-crossing time. A sampled controller holds its previous command until the next update.

—

The initial condition matters

Different offsets select different laws.

The current offset is the parameter. It selects a critical region and therefore a different stored control law.
